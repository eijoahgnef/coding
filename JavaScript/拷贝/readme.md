# 浅拷贝
对基础数据而言，就是对其赋值，且复制后的变量和原变量互不影响，对于引用数据而言，浅拷贝只是复制栈上所指向堆上的地址。

# 深拷贝
对引用数据类型复制，改变数据不会影响原数据。

# 赋值
A=B，A为`左值`，B为表达式

## 深拷贝的实现
- 基本数据类型是否能拷贝？
- 键和值都是基本类型的`普通对象`是否能拷贝？
- Symbol 作为对象的key是否能拷贝？
- Date 和 RegExp 对象类型是否能拷贝？
- Map 和 Set 对象类型是否能拷贝？
- Function 对象类型是否能拷贝？
- 对象的原型是否能拷贝？
- `不可枚举属性`是否能拷贝？
- `循环引用`是否能拷贝？
